Hero Data Structure


            
            

                
Static Hero Data Structure

                
Dependent on-chain data table contract: ****

                
            

                    

                        
1. Talent Skills Table

                        
Default includes four talents {Spring, Summer, Autumn, Winter}, each with five skills and their data characteristics.

                    

                    
                    

                        
2. Race Table

                        
Default includes five races {Human, Elf, Dwarf, Orc, Undead}.

                


                        

                        
3. Class Table

                        
Default includes five classes {Warrior, Mage, Archer, Rogue, Priest}.

                        

                    
                        

                        
4. Attributes Table

                        
Default includes four attributes {Agility, Attack, Health, Defense}.

                        

                    
                        

                        
5. Equipment Table

                        
Default includes three equipment types {Weapon, Armor, Accessory}, can correspond to different contract addresses.

    


                    

                        
6. Item Table

                        
Default includes one item type {Item}, can correspond to different contract addresses.

                    

                    
                    

                        
7. Daily Energy and Points Limit

                        
Default daily energy is 100, points earning limit is 1000. Entering a dungeon consumes varying amounts of energy.

                    


                    

                        
Example JSON Structure:

                        
{
    "name": "Hero Name",
    "race": "Human",      // default is Human
    "gender": "Male",     // default is Male
    "level": 1,          // default is 1
    "energy": 100,       // default is 100, consumed when entering dungeons (30-100)
    "skills": {
        "Spring": [1,0,0,0,0],
        "Summer": [0,1,0,0,0],
        "Autumn": [0,0,1,0,0],
        "Winter": [0,0,0,1,0]
    },
    "equipment": ["Weapon 1", "Armor 1", "Accessory 1"]  // default is []
}
